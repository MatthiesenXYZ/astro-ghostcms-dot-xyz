## Install

The Best way to install Astro-GhostCMS is from a fresh install.  To get started run the following

<Tabs>
  <TabItem label="npm">
    ```sh
    npm create astro@latest
    # Create Empty Install with standard typescript
    # Then Delete entire `pages` folder under `/src/`
    ```

Then run the following to install the package!

    <Tabs>
      <TabItem label="Astro `add` method">
        ```sh
        # Astro Quick mode
        npx astro add @matthiesenxyz/astro-ghostcms
        ```
      </TabItem>
      <TabItem label="Manual pkg Install">
        ```sh
        # Manual Method
        npm install @matthiesenxyz/astro-ghostcms
        ```
      </TabItem>
    </Tabs>

  </TabItem>
  <TabItem label="pnpm">

Then run the following to install the package!

Then set your `astro.config.mjs` to look like this:

```ts frame="code" title="astro.config.mjs"
import { defineConfig } from "astro/config";
import GhostCMS from '@matthiesenxyz/astro-ghostcms';

// https://astro.build/config
export default defineConfig({
  site: "https://YOUR-DOMAIN-HERE.com"
  integrations: [GhostCMS({
    // You can either set your ghostURL here or as a `.env`/environment variable 
    // THIS WILL TAKE PRIORITY
    ghostURL: "https://ghostdemo.matthiesen.xyz",
  })],
});
```

## Created Routes

The routes are the same as a standard Ghost Blog so you can migrate to Astro easily.

| Route                 | Content                                   |
| --------------------- | ----------------------------------------- |
| `/`                   | Homepage with recents/features Blog Posts |
| `/404`                | 404 Page                                  |
| `/[slug]`             | Post or Page                              |
| `/author/[slug]`      | Author page with related posts            |
| `/authors`            | All the authors                           |
| `/tag[slug]`          | Tag page with related posts               |
| `/tags`               | All the tags                              |
| `/archives/[...page]` | All the posts, paginated                  |
| `/rss.xml`            | All the posts, in a FEED                  |
